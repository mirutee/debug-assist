Como funciona

3 passos. Diagnóstico na hora.

Não precisa de IA gigante rodando localmente. Só um endpoint e a sua chave.

01

Receba o erro

Captura qualquer erro no seu app — um hydration error do React, um SQL lento, uma chamada de API que retornou 401.

02

Envie para o DevInsight

Uma chamada POST com o tipo do erro, a mensagem e contexto opcional. Funciona de qualquer linguagem ou framework.

03

Receba o diagnóstico

Problema identificado, causa explicada em português, nível de severidade e sugestões de correção prontas pra ação.

Integração

Pronto em 2 minutos

Instale o SDK e comece a diagnosticar. Funciona de dentro do catch de qualquer função.

JavaScript 
import { reportError } from 'devinsight-sdk';

// No catch do seu componente React:
try {
  const data = await fetch('/api/user');
} catch (err) {
  const diagnostico = await reportError({
    tipo: 'request_error',
    mensagem: err.message,
    status: 401
  }, process.env.DEVINSIGHT_KEY);

  // → "Token ausente, inválido ou expirado"
  console.log(diagnostico.causa);
}
